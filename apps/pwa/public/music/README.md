# Cebby Wrapped - Music Setup

## Music Folder Structure

Place your background music files in the `/public/music/` directory:

```
apps/pwa/public/
└── music/
    ├── chill-lofi.mp3
    ├── upbeat-techno.mp3
    └── ambient-focus.mp3
```

## Supported Formats
- MP3 (recommended)
- OGG
- WAV

## Recommendations

### 1. Chill Lofi
- **Vibe**: Relaxed, nostalgic, cozy
- **BPM**: 80-100
- **Good for**: Reflecting on the year, calm storytelling
- **Suggested tracks**: Lo-fi hip hop beats, jazzy piano loops
- **Example sources**: 
  - YouTube Audio Library (search "lofi")
  - Free Music Archive (Creative Commons)

### 2. Upbeat Techno
- **Vibe**: Energetic, celebratory, modern
- **BPM**: 120-140
- **Good for**: Achievement celebrations, dynamic stats
- **Suggested tracks**: Electronic house, synth-wave
- **Example sources**:
  - Incompetech (Kevin MacLeod)
  - Bensound

### 3. Ambient Focus
- **Vibe**: Ethereal, atmospheric, inspiring
- **BPM**: 60-90
- **Good for**: Community moments, thoughtful slides
- **Suggested tracks**: Ambient pads, cinematic soundscapes
- **Example sources**:
  - Purple Planet Music
  - Incompetech

## Finding Royalty-Free Music

### Free Sources:
1. **YouTube Audio Library** - https://www.youtube.com/audiolibrary
2. **Free Music Archive** - https://freemusicarchive.org
3. **Incompetech** - https://incompetech.com
4. **Purple Planet** - https://www.purple-planet.com
5. **Bensound** - https://www.bensound.com

### Paid (High Quality):
1. **Epidemic Sound** - https://www.epidemicsound.com
2. **Artlist** - https://artlist.io
3. **Musicbed** - https://www.musicbed.com

## File Size Guidelines
- Keep files under 5MB each for fast loading
- Use 128kbps MP3 for web optimization
- Consider using short loops (30-60 seconds) that repeat seamlessly

## Testing
After adding music files:
1. Navigate to `/wrapped/2024`
2. Click the music icon (top right)
3. Select each track to test playback
4. Verify volume is balanced across tracks

## Note
Music will auto-start when available, but may be blocked by browser autoplay policies. Users can manually select a track from the music selector.
